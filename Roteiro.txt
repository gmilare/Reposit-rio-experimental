Resumo da sequência de comandos git utilizados no experimento


1. Criar novo repositório, selecionando "New repository" na página do github

1.1. Definir próprio usuário como proprietário
1.2. Nomear repositório
1.3. Informar a descrição do repositório
1.4. Selecionar repositório de maneira pública
1.5. Adicionar README.md
1.6. Selecionar linguagem python
1.7. Selecionar MIT License

2. Criar cópia do repositório na máquina local utilizando o comando git clone no prompt de comando do Windows

git clone https://github.com/gmilare/Repositorio-experimental

3. Listar os arquivos da pasta corrente, verificando a existência da pasta 'Repositorio-experimental', utilizando o comando dir

dir

4. Para alterar a pasta corrente para a pasta 'Repositorio-experimental", utilizar o comando cd

cd Repositorio-experimental

5. Para obter a lista completa de arquivos, inclusive os ocultos, utilizar o comando dir /a

dir /a 

6. Verificando o status do repositório com o comando git status

git status
